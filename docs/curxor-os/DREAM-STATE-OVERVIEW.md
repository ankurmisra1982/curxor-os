## Operator experience (dream state)

### Day zero
Operator plugs in CurXor. FRE provisions the desk — which Claws to enable, intelligence mode (local / frontier / auto), household context. **The Forge** mints additional Claws from intent, photos, or live vision. No assembly. No weekend duct-tape.

### Day one
**Capital**, **Creator**, and **Outreach** run while the operator sleeps — within rules the operator set. Trades and posts cross **egress** only after explicit skill intent; cognition stays local. Flight Command is the command surface, not a browser tab full of chats.

### Day thirty
**Kin** knows the household. **Vital** knows the body. **Signal** pairs robots, wearables, glasses, VR, and home devices — the desk **conducts** interfaces instead of multiplying disconnected apps. **Claw Cafe** is the spatial home: handshakes, ascension, shared presence. Claws share context through **CCP** without the operator re-explaining everything.

### Day forever
The system **grows with the operator** — more Claws, more devices, more surface area — same OS, same desk, same ownership. No new subscription every time a workflow appears.

## Sovereignty principles (non-negotiable)

1. **Local inference by default** — cognition on `127.0.0.1`; cloud is opt-in frontier, not the product.
2. **Physical egress boundary** — command port for operators; egress port for outbound agent action. Unplug egress → outbound stops; thinking can continue.
3. **No LLM-initiated trades or posts** — skills → mesh → Python bridges after operator intent.
4. **Own the stack** — hardware + OS + agents; `$3,999 once` economics vs API rent (operate plane).
5. **Operator is sovereign** — Go Live · approvals · consent scopes on CCP; Master AI **patron**, not autopilot over the operator.

## Thesis: the orchestrator on your desk

The category mistake is chasing the next **device** — pin, glasses, pod, humanoid — without a conductor. Devices drift. CurXor’s bet: **the desk** is where intelligence, policy, and outbound action meet.

- **Signal** delivers to surfaces (glance · VR · room · robot).
- **Claw Cafe** gives spatial presence to the team.
- **Master AI** (horizon) sits above as patron — not a replacement for the ten, not a chat SaaS tab.
